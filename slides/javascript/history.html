JS the history

  • Created (in 10 days) by Brendan Eich in 1995.
  • Inspired by (among others):
    • Java (syntax)
    • Scheme (first-class functions and closures)
    • Self (prototypal inheritance)
  • The name JavaScript was chosen to market Java.
  • Standardisation started in 1996 under the name ECMAScript (ES). The TC39 process governs how ES features are designed.
