JS

type coercion

Implicit conversion (1/4)

  • Because JS is a loosely-typed language, implicit conversion of types is allowed: 
    
        '2' * '3';    // 6
        'a' + 1;      // 'a1'
        true + false; // 1
  • Any type in JS can be converted: 
    
        {}+[]+{}+[1]; // '0[object Object]1' ☹️
  • All operators (except ===) trigger type coercion: 
    
        1 >= '1'; // true
        1 + '1';  // '11'
        1 / '1';  // 1

String coercion (2/4)

  • When any operand is a string, the + operator triggers implicit type coercion: 
    
        '' + 1        // '1'
        true + ' dat' // 'true dat'
  • Use String() for explicit coercion: 
    
        String(1)     // '1'
        String(null)  // 'null'

Boolean coercion (3/4)

  • In JS these values are falsy: 
    
            Boolean('');         // false
            Boolean(0);          // false
            Boolean(NaN);        // false
            Boolean(null);       // false
            Boolean(undefined);  // false
            Boolean(false);      // false
    Everything else is truthy: 
    
            Boolean('a');      // true
            Boolean(1);        // true
            Boolean({});       // true
            Boolean([]);       // true
  • For implicit conversion, use if or a logical operator: 
    
            if (1) { … }
            
            0 && 1;         // 0 ⚠️
            0 || 1;         // 1 ⚠️
            0 ? 1 : 2       // 2 ⚠️
            ES2020
            0 ?? 1;         // 0
            null ?? 1;      // 1
            undefined ?? 1; // 1
    Use Boolean() or ! for explicit coercion: 
    
            Boolean(1)      // true
            !1              // false
            !!1             // true

Number coercion (4/4)

  • It's complicated… Almost all operators implicitly convert to number: 
    
            // comparison operators
            2 > '1'       // true
            false <= '1'  // true
            // arithmetic operators
            '3' * true    // 3
            4 / null      // Infinity
            4 + ''        // '4' ⚠️
            // unary + operator
            +'123'        // 123
            // loose equality operators
            '' == 0       // true
            2 != '2'      // false
            '' == 0       // true
            // except:
            '' == '0'     // false ⚠️
            null == 0     // false ⚠️
            null == false // false ⚠️



    Explicit conversion:

    
            // booleans
            Number(true)      // 1
            Number(false)     // 0

            // strings
            Number('1')       // 1
            Number(' -2 ')    // -2
            Number('3px')     // NaN
            Number.parseInt('3px', 10)    // 3
            Number.parseFloat('3.14 abc') // 3.14

            // other primitives
            Number(null)      // 0
            Number(undefined) // NaN ⚠️
